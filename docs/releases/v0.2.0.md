## Extensions, member photos & documentation

### Added
- Member profile photos — upload or generate initials avatar from the member profile
- Custom entities API — per-gym record types without schema migrations
- Custom analytics API — saved metrics (COUNT/SUM) over members, payments, attendance, custom entities
- Documentation site: product guides (`/docs`), developers (`/developers`), architecture (`/docs/architecture`)

### API routes
- `POST /api/members/photo` — multipart upload
- `POST /api/members/photo/generate` — DiceBear avatar
- `GET|POST /api/custom-entities`
- `GET|POST /api/custom-entities/[id]/records`
- `GET|POST /api/custom-analytics`
- `GET /api/custom-analytics/[slug]/run`

### Upgrade
```bash
npx prisma migrate deploy
npm run openapi:generate   # if you vendor openapi.json
```
